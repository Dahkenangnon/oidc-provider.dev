## Implemented Specifications

<Aside type="note" title="Table Update Schedule">
  This specifications table is **manually updated weekly** or whenever the upstream [node-oidc-provider README](https://github.com/panva/node-oidc-provider#implemented-specs--features) changes. Not all features are enabled by default—see [configuration](/configuration/features) to enable them.
</Aside>

- Opaque
- [JSON Web Token (JWT) Profile for OAuth 2.0 Access Tokens](https://www.rfc-editor.org/rfc/rfc9068.html)
